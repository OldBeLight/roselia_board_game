## 流程

在本游戏中，玩家将扮演Roseia中的成员角色，奔波于世界各地。通过达成【Live挑战】，赢得亚洲巡演各地粉丝的喜爱，称霸该巡演地，来减少自己的【角色不可崩点数】。最终成为所有人中【角色不可崩点数】最少的一位，在顶点狂乱绽放，来达成游戏胜利。

所有角色都在【起点】，玩家选择想要扮演的角色，然后投掷骰子决定先发顺序，点数大的玩家先行动，之后的行动轮次按照这个顺序决定。若掷出相同点数，相同点数的人需要重新投掷直到点数不相同。

玩家行动时通过掷一个骰子决定【步行点数】，在规定路径上的任意方向行走与【步行点数】相同的格子数量，若停在特殊指令的【格子】，则立即执行格子上的指令。

当玩家正好走至任何没有被挑战成功的【Live挑战】时，即可进行挑战。挑战成功可获得此【巡演地】，每拥有一个巡演地可以减少30点【角色不可崩点数】。 当两名玩家正好走至相同格子时，需要掷骰子进行点数pk，点数大的一方获得对方一个巡演地。若败方没有任何巡演地，则胜方不能获取任何东西。 当6个【Live挑战】全部完成时，最先去到【顶点】的玩家减少50点【角色不可崩点数】，游戏结束。结算【角色不可崩点数】最少的玩家为游戏胜者。 

可追加规则：玩家需要扮演自己所选的角色，如果ooc了，【角色不可崩点数】+1

## 术语解释

### 角色骰子

一共两个骰子，可以拿其中一个充当【角色骰子】 掷出1代表：【冰川纱夜】 掷出2代表：【宇田川亚子】 掷出3代表：【凑友希那】 掷出4代表：【白金燐子】 掷出5代表：【今井莉莎】 掷出6代表：【指名】，即由掷骰人决定一个角色 ~~角色顺序参考手游，不代表作者偏好谁是1~~
### 一方通行

只能按照箭头方向前进
### 卡片

抽取一张卡片并立即生效。去这里抽卡：[https://roselia-board-game.onrender.com/card.html](https://roselia-board-game.onrender.com/card.html)
### 根据角色骰子进行交换

掷【角色骰子】来决定人选。立即交换两位玩家的位置，交换后两位玩家所在格子有特殊指令（例如【卡片】【传送门】以及其他）并不执行
### 传送门

一共有4个传送门，走到传送门可以直接传送到任意一个其他传送门，传送不消耗【步行点数】
### 根据角色骰子强制行动

掷【角色骰子】来决定人选。再掷一次骰子决定【步行点数】
### 骰子翻倍

下回合行动时，掷两个骰子，点数相加为本次【步行点数】
### 把某人邀请到武士道

掷【角色骰子】来决定人选。如果掷出自己，回到武士道的起点
### 和某人一起友好地获得角色不可崩点数

掷【角色骰子】来决定人选，再掷一次骰子决定要扣除的【角色不可崩点数】。如果掷出自己，扣除双倍的【角色不可崩点数】
### 顶点

当6个【Live挑战】全部完成时，最先去到【顶点】的玩家减少50点【角色不可崩点数】，游戏结束。结算【角色不可崩点数】最少的玩家为游戏胜者。若仍然存在【Live挑战】没有完成，则不会发生任何事情。
## Live挑战

大阪：模仿Roseia中之人全员笑声，若其他玩家根据事实考量，超过半数觉得不像，则挑战失败

新加坡：打开手机的计时器并闭眼，数10秒钟并停止计时，如果时间范围落在9-11秒则挑战成功。若闭眼时想要按停止计时却不小心按漏了，不消耗回合重新挑战

首尔：fc一首26级的Roseia歌曲，如果挑战失败，下次所有人进入此挑战所需歌曲等级-1

东京：通过【角色骰子】决定一名非挑战者玩家，b站打开[BV11YZuYsEfS](https://www.bilibili.com/video/BV11YZuYsEfS)，选择一首歌曲的单乐器分轨，让挑战者猜歌曲名称

上海：说出 [自己角色的身高、生日] 或者 [中之人血型、身高、生日] 的其中两个。或者正确拼写出 Roseia 2025 亚巡的名称。或者唱Roseia的歌。或者说出你推的10个优点，你推不一定是你玩的角色因为你抢不到。

台北：把【角色名称】替换成自己扮演的角色的名称（声优名称也行），念以下羞耻台词。超过半数玩家不满意表现则挑战失败
