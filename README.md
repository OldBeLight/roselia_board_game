# 规则书

## 前言

此游戏仿制Rosenchor北海道幕间桌游，规则有些许改动，游玩的时候也可以根据实际情况自己修改规则，因为本人并没有测试过游戏数值，大概率是不平衡的。刚好亚巡也是6个地点真是太好了。如果有人希望自印给自己玩或者自己朋友玩，是完全可以的，别大规模印（超过20张）就行。想自己修改内容然后印给自己或朋友玩也是完全可以的，前提是修改内容不涉及侮辱和造谣Roselia二次元角色以及其配音演员。

## 流程

在本游戏中，玩家将扮演Roseia中的成员角色，奔波于世界各地。通过达成【Live挑战】，赢得亚洲巡演各地粉丝的喜爱，称霸该巡演地，来减少自己的【角色不可崩点数】。最终成为所有人中【角色不可崩点数】最少的一位，在顶点狂乱绽放，来达成游戏胜利。

所有角色都在【起点】，玩家选择想要扮演的角色，然后投掷骰子决定先发顺序，点数大的玩家先行动，之后的行动轮次按照这个顺序决定。若掷出相同点数，相同点数的人需要重新投掷直到点数不相同。

玩家行动时通过掷一个骰子决定【步行点数】，在规定路径上的任意方向行走与【步行点数】相同的格子数量，若停在特殊指令的【格子】，则立即执行格子上的指令。

当玩家正好走至任何没有被挑战成功的【Live挑战】时，即可进行挑战。挑战成功可获得此【巡演地】，每拥有一个巡演地可以减少30点【角色不可崩点数】。
当两名玩家正好走至相同格子时，需要掷骰子进行点数pk，点数大的一方获得对方一个巡演地。若败方没有任何巡演地，则胜方不能获取任何东西。
当6个【Live挑战】全部完成时，最先去到【顶点】的玩家减少50点【角色不可崩点数】，游戏结束。结算【角色不可崩点数】最少的玩家为游戏胜者。
可追加规则：玩家需要扮演自己所选的角色，如果ooc了，【角色不可崩点数】+1
## 术语解释

### 角色骰子

一共两个骰子，可以拿其中一个充当【角色骰子】
掷出1代表：【冰川纱夜】
掷出2代表：【宇田川亚子】
掷出3代表：【凑友希那】
掷出4代表：【白金燐子】
掷出5代表：【今井莉莎】
掷出6代表：【指名】，即由掷骰人决定一个角色
~~角色顺序参考手游，不代表作者偏好谁是1~~
### 一方通行

只能按照箭头方向前进
### 卡片

抽取一张卡片并立即生效
### 根据角色骰子进行交换

掷【角色骰子】来决定人选。立即交换两位玩家的位置，交换后两位玩家所在格子有特殊指令（例如【卡片】【传送门】以及其他）并不执行
### 传送门

一共有4个传送门，走到传送门可以直接传送到任意一个其他传送门，传送不消耗【步行点数】
### 根据角色骰子强制行动

掷【角色骰子】来决定人选。再掷一次骰子决定【步行点数】
### 骰子翻倍

下回合行动时，掷两个骰子，点数相加为本次【步行点数】
### 把某人邀请到武士道

掷【角色骰子】来决定人选。如果掷出自己，回到武士道的起点
### 和某人一起友好地获得角色不可崩点数

掷【角色骰子】来决定人选，再掷一次骰子决定要扣除的【角色不可崩点数】。如果掷出自己，扣除双倍的【角色不可崩点数】
### 顶点

当6个【Live挑战】全部完成时，最先去到【顶点】的玩家减少50点【角色不可崩点数】，游戏结束。结算【角色不可崩点数】最少的玩家为游戏胜者。若仍然存在【Live挑战】没有完成，则不会发生任何事情。

## 卡片

【所以说武士道就是恶魔之路】去武士道(2张)

【罪孽深重的女人】所有人一起去武士道(1张)

【小猫咪】选择一位玩家，每句话的结尾都要带上喵，如果已经抽到类似卡片，则追加而不是替换。若违反规定，角色不可崩点数+1(1张)

【大小姐】选择一位玩家，每句话的结尾都要带上desuwa，如果已经抽到类似卡片，则追加而不是替换。若违反规定，角色不可崩点数+1(1张)

【咋瓦鲁多】选择一位玩家，跳过此玩家的下一回合(2张)

【好强的压】选择一位玩家，摧毁此玩家拥有的其中一个【巡演地】，此【巡演地】需要重新站在格子上并且【Live挑战】成功才能获得。若有多个巡演地，由抽中卡片的玩家挑选一个摧毁(1张)

【以牙还牙】最近一次被其他玩家施加的【小猫咪】【大小姐】【咋瓦鲁多】【好强的压】效果反弹到该玩家身上。如果负面效果是【小猫咪】【大小姐】，解除抽卡玩家身上的此效果（1张）

【角色不可崩点数减少5点】(2张)

【千载一遇】传送到离没有挑战成功的【Live挑战】相距一格的距离并结束本回合，如果所有【Live挑战】已全部完成，则传送到离【顶点】相距一格的距离。可以任意挑选格子。(1张)
